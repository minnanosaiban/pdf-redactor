# pdf-redactor


## このアプリについて

- スキャンPDF・サーチャブルPDFの墨消しをするアプリです。オフライン版も用意しています。
- **処理はすべてブラウザ内で完結し、PDFはサーバーに送信されません。**
- 公開は Cloudflare Pages を利用しています。

## 免責事項

- 本ツールは無保証で提供されます（現状有姿）。なお、本アプリのコードは GitHub に公開しています。
- 処理はお使いのブラウザ内で完結しPDFは外部送信されませんが、ご利用環境に起因する情報漏えいについて作成者は責任を負いません。
- 本ツールの利用により生じたいかなる損害についても作成者は責任を負わず、本ツールは法的助言を提供するものではありません。
- 墨消しは各ページを画像化して黒く塗りつぶす方式です。消し忘れ・塗り不足を含め、出力結果の最終確認は利用者の責任で行ってください。

## 仕組み（ブラウザ完結）

- `pdf.js` … PDFの各ページを画面に表示します。
- ドラッグで黒塗り範囲を指定します（枠クリックで削除、元に戻す、ページ単位でのクリアも可能です）。
- 「墨消しを適用」を押すと、**各ページを画像化して黒塗りを画素に焼き込み**、`pdf-lib` で新しいPDFを生成します。
  - 出力は画像PDFになるため、**元の文字・画素は一切含まれません＝確実に墨消しできます**。
  - 元PDFのメタデータも引き継がれません。
- React / Next.js は使用していません（素のHTML＋バニラJSで実装しています）。

## オフライン版

- `pdf.js` / `pdf-lib` を含めて全部を1枚のHTMLに埋め込んでおり、**保存後はネット接続なしで動作します**。
- ソースはテキストエディタでそのまま読めるため、処理内容をご自分の目で確認いただけます。
- `scripts/build-offline.js` が `public/index.html` + `public/app.js` + `vendor/*.js` から自動生成します
  （ロジックは公開版と共通で、手作業での二重管理はしていません）。
- `npm run build:offline` で再生成できます（`npm run dev` / `npm run deploy` 実行時にも自動で実行されます）。
- `vendor/` の中身を更新したとき（pdf.js / pdf-lib のバージョンアップ等）は、再生成を忘れないようにしてください。
- 制約：`file://` で直接開くと、pdf.js側の実ワーカー起動がブラウザにブロックされ、
  自動的にメインスレッド処理へフォールバックします（動作はしますが、ページ数が多いと重くなる場合があります）。
  実ワーカーを使いたい場合は、下記のローカルサーバー経由で開いてください。

## ローカルで動かす

```bash
npm install          # wrangler を入れる
npm run dev          # http://localhost:8788 で確認（file:// では pdf.js ワーカーが動かない）
```
`python -m http.server -d public 8000` でも構いません（http で配信してください）。

## 公開（Cloudflare Pages）

```bash
npm run deploy       # wrangler pages deploy
```
本人・関係者限定にする場合は、Cloudflare Access（Google ログイン）を前段に設定してください。

## 運用上の注意

- **墨消しを適用する前には、必ず目視でレビューしてください**（自動検出ではなく手動指定のため、消し忘れにご注意ください）。
- 出力後は、別途PDFを開いて**墨消し漏れ・下地露出がないか確認してください**。
- 入力・出力PDFは `.gitignore` で除外済みです。**PIIを含むPDFはコミット・pushしないでください。**
